# Restaurant Site Template

This is a minimal Next.js app template for restaurant sites.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Add your restaurant data:
   - Place `config.json` in `data/` directory
   - Place `menu.json` in `data/` directory

3. Run the development server:
   ```bash
   npm run dev
   ```

## Data Files

- `data/config.json` - Restaurant configuration
- `data/menu.json` - Menu data

See the core platform documentation for the schema of these files.
